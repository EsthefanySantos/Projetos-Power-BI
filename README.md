## 💻 Sobre o projeto
Este dashboard foi desenvolvido no Power BI com o objetivo de apresentar uma visão geral das vendas, permitindo analisar faturamento, quantidade vendida, produto mais vendido e distribuição geográfica.

O projeto foi desenvolvido com o objetivo de aperfeiçoar meus conhecimentos em Power BI, com base em conteúdos do canal Hashtag Treinamentos no YouTube.
